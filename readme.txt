== Description ==

A Widget that links to posts and pages matching search engine terms used to find the page. For instance if the page displayed is where the user landed after clicking a Google search result link the widget tries to inform about more reading on the site that might interest the user.

== Installation ==

1. Upload `search-on-search.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Drag it onto your sidebar from Admin->Themes->Widgets

== Frequently Asked Questions ==

= Does it support all search engines? =

No, just the more common ones. Please let me know if your fav search engine is missing and I probably add it.
